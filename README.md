# Project_Inbio_TBC_MachineLearning
  Penelitian ini merupakan self project saat magang mandiri dengan INBIO dengan judul project "MACHINE LEARNING-BASED DISCOVERY OF ANTI-TUBERCULOSIS AGENTS FROM INDONESIAN MEDICINAL PLANTS THROUGH COMPOUND FINGERPRINTING"

Penelitian ini berfokus untuk skrining senyawa potensial dengan Mechanism of Action yang unik dengan mempelajari struktur senyawa aktif bersifat anti TBC sebagai training data dalam ML. Penelitian ini masih on progress dan dalam draft penulisan
